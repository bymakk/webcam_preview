# Webcam Preview

Простой статический дашборд для мониторинга живых превью моделей
**Chaturbate** и **Stripchat**. Плитки 16:9, перетаскивание, переключение
видео/картинка из контекстного меню, всё хранится в браузере (localStorage).

> Деплоится как обычная статика — **GitHub Pages**. Без сборки, **без бэкенда и без прокси**.
> Видео берётся из встроенных плееров-виджетов в `<iframe>` (как у striphours).

---

## Возможности

- ➕ Плюс по центру → добавить логин Chaturbate и/или Stripchat (одна модель может быть на обеих площадках).
- 🎯 По умолчанию: **только Chaturbate → картинка**, **только Stripchat → видео**, **обе → выбор** режима и площадки.
- 🖼️ Картинка — это снимок (`thumb.live`), обновляется по таймеру (интервал в настройках), а не живой MJPEG-поток.
- 🧱 Плитки **16:9**, число колонок в сетке меняется ползунком (1–8).
- 🖱️ **Перетаскивание** плиток мышью (drag & drop). На тач-экранах и с клавиатуры — пункты меню *Сдвинуть влево/вправо* (плитка фокусируется, меню открывается клавишей контекста / Enter).
- 🖼️ / 🎥 **Правый клик** по плитке → меню: режим *Картинка ⇄ Видео*, выбор площадки, перемещение, **Редактировать**, открыть трансляцию, удалить.
- 🔗 При наведении — логины модели **ссылками** на трансляцию + иконка редактирования.
- 💾 Экспорт/импорт конфигурации, всё сохраняется локально.

## Как работают превью (без прокси)

| Источник | Как |
|---|---|
| **Chaturbate — картинка** | `<img>` снимок `thumb.live.mmcdn.com`, обновляется по таймеру. |
| **Chaturbate — видео** | `<iframe>` официальный плеер `chaturbate.com/embed/{логин}/`. |
| **Stripchat — видео/картинка** | `<iframe>` плеер-виджет `creative.mavrtracktor.com/widgets/Player?modelName={логин}` — принимает логин, сам резолвит и расшифровывает поток. |

Видео-виджеты в `<iframe>` сделаны с `pointer-events: none`, поэтому наши
наведение/контекстное меню/перетаскивание работают поверх, а плеер играет под ними.

> ⚠️ **Chaturbate-видео**: встроенный плеер может показать заставку «18+»
> (из-за блокировки сторонних cookie в `<iframe>`). Поэтому для Chaturbate
> надёжнее режим **Картинка** (он и стоит по умолчанию). Stripchat-видео играет
> сразу. Виджеты — это партнёрские плееры площадок (как на striphours.com).

## 1. Запуск фронтенда на GitHub Pages

1. Запушьте содержимое репозитория (включая `index.html`, `app.js`, `styles.css`, `.nojekyll`).
2. **Settings → Pages → Source: Deploy from a branch**, ветка `main`, папка `/ (root)`.
3. Откройте `https://<user>.github.io/webcam_preview/`.

Локально: любой статический сервер, например
```bash
python3 -m http.server 8000
# затем http://localhost:8000
```
(Можно открыть и просто `index.html` файлом — Chaturbate-картинки заработают.)

## 2. (Необязательно) Прокси — только для статуса online/offline у Chaturbate

Прокси **не нужен** для превью. Он лишь добавляет точный индикатор LIVE/OFF в
режиме картинки Chaturbate (`thumb.live` отдаёт 200 даже для оффлайна, поэтому
без прокси точка статуса скрыта). Если хочется — разверните Worker:

```bash
npm i -g wrangler        # или npx wrangler ...
wrangler login
wrangler deploy          # использует wrangler.toml + worker.js
```
и вставьте URL в **Настройки → URL прокси**. Можно полностью игнорировать.

## Структура

```
index.html     разметка (тулбар, сетка, модалки)
styles.css     тёмная тема, плитки 16:9, оверлеи
app.js         вся логика: состояние, превью, DnD, контекстное меню, модалки
worker.js      Cloudflare Worker: /health и /resolve (CB + SC)
wrangler.toml  конфиг деплоя воркера
```

## Заметки

- Данные (модели, настройки) — только в `localStorage` вашего браузера.
- Это инструмент мониторинга **публичных** превью; контент 18+ принадлежит площадкам.
